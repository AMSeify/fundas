Contributing to Fundas
======================

We welcome contributions to Fundas! Whether you're fixing a bug, adding a new feature, or improving documentation, your help is appreciated.

Getting Started
---------------

1. **Fork the Repository**
   Start by forking the `fundas repository`_ on GitHub.

2. **Clone Your Fork**
   Clone your forked repository to your local machine:

   .. code-block:: bash

      git clone https://github.com/YourUsername/fundas.git
      cd fundas

3. **Create a Branch**
   Create a new branch for your feature or bug fix. Use a descriptive name:

   .. code-block:: bash

      git checkout -b feature/my-new-feature

Development Workflow
--------------------

- **Install Dependencies**: Ensure you have all the necessary development dependencies installed.
- **Write Code**: Make your changes, following the existing code style.
- **Add Tests**: If you add a new feature, please include tests to verify its functionality.
- **Ensure Tests Pass**: Run the test suite to make sure your changes don't break existing code.

Submitting a Pull Request
-------------------------

Once your changes are ready, push your branch to your fork and open a pull request against the `main` branch of the official repository.

.. _fundas repository: https://github.com/AMSeify/fundas
